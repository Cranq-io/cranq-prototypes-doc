# Paths joiner

_[io/file/{$name}]_

---

Joins an array of path parts to a common base

---

__Namespace__: io/file

### Input ports:

* __base__: ` string `

    The base path portion


* __children__: ` string[] `

    The array of path parts to join onto the base

### Output ports:

* __paths__: ` any `

    The resulting paths
