# Paths joiner

_[io/file/Paths joiner]_

---

Joins an array of path parts to a common base<br>

---

### Input ports

* __base__: ` string `

    The base path portion<br>


* __children__: ` string[] `

    The array of path parts to join onto the base<br>

### Output ports

* __paths__: ` any `

    The resulting paths<br>
