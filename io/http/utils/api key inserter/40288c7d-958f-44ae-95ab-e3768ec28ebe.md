# API key inserter

_[io/http/utils/API key inserter]_

---

Inserts API key into a HTTP request header.<br>

---

__Keywords__: authorization, request, headers

### Input ports

* __headers__: ` {string: string} `


* __API key__: ` string `

### Output ports

* __headers__: ` ({string: string} and {"X-Api-Key" :string}) `
