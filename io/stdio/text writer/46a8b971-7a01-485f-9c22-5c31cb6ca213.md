# Text writer

_[io/stdio/{$name}]_

---

Writes a text to standard output.

Example:
1. "hello world"@0 received on `line`
2. "hello world" is written to the standard output

---

__Namespace__: io/stdio

### Input ports:

* __text__: ` string `

    The text to be written to the standard output.
    
    
    Example: "hello world"
