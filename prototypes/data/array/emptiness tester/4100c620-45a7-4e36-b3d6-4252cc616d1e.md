# Emptiness tester

_[data/array/Emptiness tester]_

---

Tests whether array is empty.<br>
<br>
Example:<br>
1. []@0 received via `array` <br>
2. true@0 sent via `empyty`<br>

---

__Keywords__: empty

### Input ports

* __array__: ` any[] `

    Receives array to be tested.<br>
    <br>
    Example:<br>
    []<br>

### Output ports

* __empty__: ` boolean `

    Sends whether the `array` does not contain any element.<br>
    <br>
    Example:<br>
    true<br>
