# Step mapper

_[data/array/Step mapper]_

---

Almost identical to `data/array/Mapper`, except it does not release the next item before receiving the previous mapped item.<br>

---

__Keywords__: foreach, iterate, project, loop, async

### Input ports

* __array__: ` any[] `


* __mapped item__: ` any `

### Output ports

* __mapped__: ` any[] `


* __item__: ` any[][number] `


* __done__: ` any `


* __bounced__: ` any `
