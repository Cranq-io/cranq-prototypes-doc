# Sorter

_[data/array/Sorter]_

![icon](</assets/icons/cbb85c56-3c8f-4e5e-afdd-a9dd9e84385d.png>)

---

Sorts an array according to the specified reference array (`params`["order"]).<br>

---

### Input ports

* __array__: ` any[] `


* __params__: ` {"order": any[], optional "direction": (1 or -1)} `

### Output ports

* __sorted__: ` any[] `
