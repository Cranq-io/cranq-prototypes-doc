# Path deleter

_[io/file/Path deleter]_

![icon](</assets/icons/b63b05e2-5e2f-45a1-a3e3-10a749d05b96.png>)

---

Deletes the specified path recursively, and forced.<br>

---

### Input ports

* __path__: ` string `

    Deletes all folders on the specified path from the filesystem.<br>

### Output ports

* __success__: ` boolean `


* __error__: ` string `
