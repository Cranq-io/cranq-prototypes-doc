# File form field header builder

_[io/http/utils/File form field header builder]_

---

Builds a header to be included in a multipart/form-data request.<br>

---

### Input ports

* __field__: 
    ```
    {"name" :string, "fileName" :string, optional "contentType" :string}
    ```

### Output ports

* __header__: ` string `
