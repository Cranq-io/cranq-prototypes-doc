# Abs

_[number/Abs]_

---

Returns the absolute value of the input number.<br>
<br>
Example:<br>
<br>
1. -23@0 is received via `number`<br>
2. 23@0 is sent via `abs`<br>

---

__Keywords__: number, operator, absolute

### Input ports

* __number__: ` number `

    Input value<br>

### Output ports

* __abs__: ` number `

    The absolute value of the input.<br>
