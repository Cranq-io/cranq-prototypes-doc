# Switch

_[flow/Switch]_

![icon](</assets/icons/0bcb4840-69c6-41d5-b1d4-2d0adcc04f37.png>)

---

Routes data received via `data` to one of the outputs matching the value in `values`, or to `other` otherwise.<br>

---

__Keywords__: switch, route, value

### Input ports

* __values__: ` string[] `


* __data__: ` any `

### Output ports

* __data__: ` any `


* __other__: ` any `
