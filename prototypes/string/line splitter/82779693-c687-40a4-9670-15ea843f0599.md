# Line splitter

_[string/Line splitter]_

---

Splits a string by lines.<br>
<br>
Example:<br>
1. 'string' receives <br>
"line1<br>
line2<br>
line3"<br>
2. 'lines' emits<br>
["line1","line2","line3"]<br>
<br>

---

__Keywords__: string split line

### Input ports

* __string__: ` string `

    Receives the string to split by lines.<br>
    <br>
    Example:<br>
    "line1<br>
    line2<br>
    line3"<br>

### Output ports

* __lines__: ` string[] `

    Emits the array of lines.<br>
    <br>
    Example:<br>
    ["line1","line2","line3"]<br>
