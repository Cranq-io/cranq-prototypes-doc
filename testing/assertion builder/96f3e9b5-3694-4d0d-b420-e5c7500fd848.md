# Assertion builder

_[testing/Assertion builder]_

---

Builds an assertion record.<br>

---

### Input ports

* __success__: ` any `


    Receives individual fields for syncing.<br>


* __actual__: ` any `


    Receives individual fields for syncing.<br>


* __expected__: ` any `


    Receives individual fields for syncing.<br>


* __type__: ` any `

### Output ports

* __assertion__: 
    ```
    {"success" :any, "actual" :any, "expected" :any, "type" :any}
    ```
