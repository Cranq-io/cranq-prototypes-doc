# Serializer

_namespace: testing/reporters/junit_

Serializes the JSON object representation of a JUnit report to XML.
Uses https://www.npmjs.com/package/junit-xml. 

### Input ports:

* __json__: ` any `

### Output ports:

* __xml__: ` any `

### Dependencies

* es6-node:
    * npm:
        * junit-xml
