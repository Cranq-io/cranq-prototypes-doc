# Serializer

_[testing/reporters/junit/Serializer]_

---

Serializes the JSON object representation of a JUnit report to XML.<br>
Uses https://www.npmjs.com/package/junit-xml. <br>

---

### Input ports

* __json__: ` any `

### Output ports

* __xml__: ` any `

### Dependencies

* es6-node:
    * npm:
        * junit-xml
