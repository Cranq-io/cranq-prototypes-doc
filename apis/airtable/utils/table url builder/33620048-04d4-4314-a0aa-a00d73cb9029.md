# Table URL builder

_namespace: apis/airtable/utils_

Builds a URL for an AirTable table.

Keywords: template

### Input ports:

* __base ID__: ` string `

    Receives the ID of an AirTable base.


* __table name__: ` string `

    Receives a table's identifier relative to an AirTable base.

### Output ports:

* __URL__: ` string `
