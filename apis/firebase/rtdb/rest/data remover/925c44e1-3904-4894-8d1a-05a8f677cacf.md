# Data remover

_[apis/firebase/rtdb/rest/Data remover]_

---

Removes the data entry identified by 'path', from a Firebase Realtime Database.<br>

---

### Input ports

* __query context__: ` any `


    idToken<br>
    dbUrl<br>


* __path__: ` any `


    Identifies data entry to be deleted.<br>
    <br>
    type: string[]<br>

### Output ports

* __done__: ` any `


* __error__: ` any `


* __query context__: ` any `


    idToken<br>
    dbUrl<br>
