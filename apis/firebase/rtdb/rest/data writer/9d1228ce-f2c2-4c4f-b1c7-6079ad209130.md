# Data writer

_[apis/firebase/rtdb/rest/Data writer]_

---

Creates or replaces a data entry  identified by 'path', in a Firebase Realtime Database.<br>

---

### Input ports

* __query context__: ` any `


    idToken<br>
    dbUrl<br>


* __path__: ` any `


    Identifies data entry to be written.<br>
    <br>
    type: string[]<br>


* __data__: ` any `

### Output ports

* __data__: ` any `


* __error__: ` any `


* __query context__: ` any `


    idToken<br>
    dbUrl<br>
