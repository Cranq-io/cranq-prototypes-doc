# Forwarder

_[flow/Forwarder]_

---

Forwards signal received via `data` (input) to `data` (output).<br>
<br>
Rarely necessary, and when it is used, it's usually to clean up nodes visually.<br>

---

__Keywords__: noop

### Input ports

* __data__: ` any `


    Receives data to be forwarded.<br>

### Output ports

* __data__: ` any `


    Sends forwarded data.<br>
