# Forwarder

_[flow/Forwarder]_

---

Forwards signal received via `data` (input) to `data` (output).  
  
Rarely necessary, and when it is used, it's usually to clean up nodes visually.  

---

__Keywords__: noop

### Input ports

* __data__: ` any `

    Receives data to be forwarded.  

### Output ports

* __data__: ` any `

    Sends forwarded data.
