# Tag extractor

_[flow/Tag extractor]_

---

Maps the signal received via `data` to its tag.  
  
Example:  
1. "A"@0 received via `data`  
2. "0"@0 sent via `tag`  

---

__Keywords__: synchronization, syncing

### Input ports:

* __data__: ` any `

    Receives the signal to extract the tag from.

### Output ports:

* __tag__: ` string `

    Sends the extracted tag.
