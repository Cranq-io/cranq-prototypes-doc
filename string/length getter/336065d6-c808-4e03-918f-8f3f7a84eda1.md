# Length getter

_[string/Length getter]_

---

Determines the length of the value received on `string`.<br>
<br>
Example:<br>
<br>
1. "foo"@0 is received via `string`<br>
2. 3@0 is sent via `length`<br>

---

__Keywords__: string, length

### Input ports

* __string__: ` string `


    The string<br>

### Output ports

* __length__: ` number `


    The length of the string.<br>
