# Regex matcher

_[string/Regex matcher]_

---

Matches a string to a non-global regular expression, and returns substrings that match the regular expression's groups.

---

__Namespace__: string

### Input ports:

* __regex__: ` string `


* __string__: ` string `

### Output ports:

* __groups__: ` (string[] or null) `


* __not found__: ` any `
