# Nodes getter

_[data/tree/Nodes getter]_

---

Retrieves multiple nodes from a tree.<br>

---

### Input ports

* __tree__: ` any `


    Data to be repeated<br>


* __paths__: ` (string or number)[][] `

### Output ports

* __nodes__: ` any[] `


* __found indexes__: ` number `


* __missing indexes__: ` any `
