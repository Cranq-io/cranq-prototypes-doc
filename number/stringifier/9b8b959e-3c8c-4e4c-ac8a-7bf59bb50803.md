# Stringifier

_namespace: number_

Serializes the given number to a string.

Example:

1. 23.45@0 is received via `number`
2. "23.45"@0 is sent via `string`

Keywords: number, string, serialize, stringify

### Input ports:

* __number__: ` number `

    The number to be converted.

### Output ports:

* __string__: ` string `

    Sends the serialized number.
