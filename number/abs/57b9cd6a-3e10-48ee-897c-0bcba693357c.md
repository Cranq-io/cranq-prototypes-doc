# Abs

_namespace: number_

Returns the absolute value of the input number.

Example:

1. -23@0 is received via `number`
2. 23@0 is sent via `abs`

Keywords: number, operator, absolute

### Input ports:

* __number__: ` number `

    Input value

### Output ports:

* __abs__: ` number `

    The absolute value of the input.
