# Command runner

_[scripting/actions/Command runner]_

---

Runs an OS command.  

---

__Keywords__: action, one-time, no-code, execute

### Input ports:

* __state__: ` any `

    Receives script state.


* __params__: 
    ```
    {"cwd" :(string or number)[], "result-path" :string, "message" :string, "command" :string}
    ```

### Output ports:

* __state__: ` any `

    Sends updated script state.
