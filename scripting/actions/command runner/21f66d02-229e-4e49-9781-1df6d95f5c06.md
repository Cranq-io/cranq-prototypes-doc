# Command runner

_namespace: scripting/actions_

Runs an OS command.

Keywords: action, one-time, no-code, execute

### Input ports:

* __state__: ` any `

    Receives script state.


* __params__: 
    ```
    {"cwd" :(string or number)[], "result-path" :string, "message" :string, "command" :string}
    ```

### Output ports:

* __state__: ` any `

    Sends updated script state.
